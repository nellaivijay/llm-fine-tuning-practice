# LLM Fine-Tuning Practice

Welcome to LLM Fine-Tuning Practice - a comprehensive educational repository for learning Large Language Model fine-tuning through hands-on labs and exercises.

## Overview

This repository provides a structured learning path for mastering LLM fine-tuning, from fundamentals to advanced topics like vision-language models. Each chapter includes:

- **Lab Documentation**: Detailed explanations and step-by-step instructions
- **Jupyter Notebooks**: Interactive notebooks with runnable code examples
- **Practice Exercises**: Real-world scenarios to test your understanding
- **Solutions**: Reference implementations for verification

## What You'll Learn

## Quick Start

### Prerequisites

- Python 3.8+
- CUDA-capable GPU (8GB+ VRAM recommended)
- 16GB RAM minimum
- 50GB free disk space

### Installation

```bash
git clone https://github.com/nellaivijay/llm-fine-tuning-practice.git
cd llm-fine-tuning-practice
./scripts/setup.sh
```

### Access Resources

- **Jupyter Notebooks**: Open `notebooks/` directory
- **Lab Documentation**: Read `labs/` markdown files
- **Solutions**: Reference implementations in `solutions/`

### Cost-Effective Learning

- **LoRA/QLoRA**: Parameter-efficient fine-tuning reduces GPU requirements

- **Interactive Notebooks**: 9 Jupyter notebooks with runnable code
- **Real-World Examples**: Practical scenarios and case studies

- **Immediate Feedback**: Run code and see results instantly

### Production-Ready Skills

## Learning Path

### For Beginners

1. Start with [Chapter 1: From Generalist to Specialist](../labs/chapter-01-generalist-to-specialist.md)
